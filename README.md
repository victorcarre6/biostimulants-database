# biostimulants-database
An open-source repository for biostimulants and bioactive compounds from plants, including data curation and cheminformatics tools.

This project aims to support researchers and companies working on sustainable agriculture by providing a structured, data-rich resource for biostimulant discovery and characterization.  

---

## Project Objectives  

- **Data Collection** ‚Äì Gather and organize information on biostimulants from scientific literature, patents, and open databases.  
- **Chemical Characterization** ‚Äì Document chemical structures, properties, and mechanisms of action for each compound.
- **Easy access** ‚Äì Provide an accessible, searchable database for biostimulants and related compounds with an API.

- **Data-Driven Insights** ‚Äì Use as a database for [Biostimulant Formulation Predictor](https://github.com/victorcarre6/biostimulant-formulation-predictor) : to predict the efficacy and optimal formulation of biostimulants via machine learning.
